TalkChef: Enhancing Culinary Experience with Adaptive Notifications

TalkChef is a Final Year Project focused on improving how home cooks interact with recipes and manage their cooking process. It is a web-based, voice-activated cooking assistant designed to make cooking more efficient, accessible, and personalized.

🚀 Key Features
🎙️ Voice-Controlled Cooking Assistant
Enables hands-free interaction using English voice commands.
⏱️ Adaptive Notifications
Helps users manage cooking time effectively with smart alerts and reminders.
🍽️ Personalized Recipe Experience
Supports customization based on user preferences and dietary needs.
🌍 Multi-Cuisine Support
Includes recipes from various cuisines such as Malay, Chinese, Indian, and Western.
📋 Step-by-Step Guidance
Displays structured cooking steps with improved clarity and usability.
🎯 Objectives
Identify and implement key requirements for a smart culinary assistant
Develop an intuitive voice-based cooking support system
Evaluate system performance to ensure usability and accuracy

💡 Purpose
This project aims to contribute to the advancement of smart kitchen technologies by providing a more interactive, intelligent, and user-friendly cooking assistant that supports better dietary management and overall cooking experience.

Setup Notes:
1. Download Xampp.
2. Start Apache and MySql in Xampp.
3. Import the sql file in this branch.
4. To open TalkChef in local, kindly enter 'http://localhost/talkchef/welcome.html' in your browser
